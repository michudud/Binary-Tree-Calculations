# Binary-Tree-Calculations

Available calculations:

- number of leaves
- largest number of edges
- comparasion of two sides of a tree

Run instructions:

- instal - npm i
- start - npm start
- run tests - npm t

Used data format:

- number of children: 0-2 - currently not protected

dataStructure = [
{
value: number,
children?: dataNode[]
}
]

dataNode =
{
value: number,
children?: dataNode[]
}
